# Hooks

Esta carpeta contiene React hooks personalizados.

## Convenciones

- Todos los hooks deben comenzar con "use" (ej: useAuth, useFetch)
- Cada hook en su propio archivo
- Documenta el propósito y parámetros del hook

## Ejemplos comunes

- useAuth - Manejo de autenticación
- useFetch - Llamadas a API
- useLocalStorage - Persistencia local

# Services

Esta carpeta contiene los servicios para comunicarse con el backend Express.

## Estructura

Cada servicio debe:

- Manejar las llamadas HTTP al backend
- Gestionar errores de API
- Transformar datos si es necesario
- Exportar funciones reutilizables

## Ejemplo

```typescript
// api.service.ts - Cliente base
// user.service.ts - Servicio de usuarios
// product.service.ts - Servicio de productos
```
